# Student Management System (C#)

A console-based Student Management System built using **C# and .NET**.  
This application allows users to perform basic **CRUD operations** (Create, Read, Update, Delete) on student records with **JSON file persistence**.

---

# Features

- Add new student
- View all students
- Search student by ID
- Update student details
- Delete student record
- Data stored locally using JSON file
